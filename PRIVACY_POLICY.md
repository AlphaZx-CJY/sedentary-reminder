# 隐私政策

**生效日期**: 2024-03-28

## 概述

「久坐提醒助手」（"我们"）非常重视用户的隐私保护。本隐私政策旨在说明本浏览器扩展如何收集、使用和保护用户信息。

## 信息收集

### 我们不收集的信息

本扩展**不收集**以下任何信息：

- ❌ 个人身份信息（姓名、邮箱、电话等）
- ❌ 浏览历史记录
- ❌ 网站访问数据
- ❌ 地理位置信息
- ❌ 设备识别信息

### 本地存储的数据

本扩展仅在用户本地浏览器中存储以下数据：

| 数据项 | 用途 | 存储位置 |
|--------|------|----------|
| 提醒间隔设置 | 记录用户的提醒时间偏好 | Chrome/Edge Storage API |
| 运行状态 | 记录计时器是否运行中 | Chrome/Edge Storage API |
| 活动统计 | 记录今日活动次数和番茄数 | Chrome/Edge Storage API |

**所有数据均存储在本地，不会上传到任何服务器。**

## 网络请求

本扩展**不进行任何网络请求**：

- 不访问远程服务器
- 不使用分析工具
- 不加载远程资源（除扩展包内文件）

## 权限说明

本扩展申请的浏览器权限及其用途：

| 权限 | 用途 |
|------|------|
| `alarms` | 设置定时提醒闹钟 |
| `notifications` | 显示系统通知提醒 |
| `storage` | 本地存储用户设置和统计 |
| `offscreen` | 离屏文档播放提示音效 |

## 第三方服务

本扩展**不使用**任何第三方服务：

- 不使用 Google Analytics
- 不使用任何追踪代码
- 不集成第三方 SDK

## 数据安全

由于所有数据均本地存储：

- ✅ 数据不会离开用户设备
- ✅ 无需担心数据泄露
- ✅ 卸载扩展即删除所有数据

## 用户权利

用户拥有以下权利：

1. **查看数据**: 所有存储数据可在扩展界面查看
2. **清除数据**: 可通过浏览器设置清除扩展数据
3. **卸载扩展**: 完全删除所有相关数据

## 政策更新

如有隐私政策更新，我们会：

1. 更新本文档
2. 在 GitHub 仓库发布更新说明
3. 必要时通过扩展通知用户
