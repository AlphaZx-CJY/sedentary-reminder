# 🎨 宣传图制作指南

Chrome Web Store 宣传图尺寸：**440x280 像素**

## 建议设计

### 方案 1: 简洁图标型

```
┌─────────────────────────────┐
│                             │
│         🧘 (大图标)         │
│                             │
│     久坐提醒助手            │
│     定时活动 · 番茄工作法   │
│                             │
└─────────────────────────────┘
```

**配色**: 
- 背景: #4CAF50 (健康绿)
- 文字: #FFFFFF (白色)

### 方案 2: 功能展示型

```
┌─────────────────────────────┐
│  ⏰ 定时提醒  🍅 番茄钟      │
│                             │
│        [大图标居中]         │
│                             │
│   久坐提醒助手              │
│   让工作更健康              │
└─────────────────────────────┘
```

## 制作工具

1. **Canva** (推荐，免费)
   - 访问 https://www.canva.com
   - 创建自定义尺寸: 440x280
   - 使用模板或自行设计

2. **Figma** (专业，免费)
   - 访问 https://www.figma.com
   - 创建 Frame: 440x280

3. **Photoshop / Sketch**
   - 新建画布: 440x280 像素

## 导出要求

- 格式: PNG 或 JPEG
- 文件名: `promo-image.png`
- 文件大小: 不超过 5MB

## 放置位置

制作完成后放在项目根目录:
```
sedentary-reminder/
├── promo-image.png   <-- 这里
├── ...
```

然后上传到 Chrome Web Store 的 "Promotional tile" 部分。
