# 📸 截图目录

此目录用于存放 Chrome Web Store 上架所需的截图。

## 所需截图规格

| 名称 | 尺寸 | 内容 |
|------|------|------|
| `screenshot-1.png` | 1280x800 或 640x400 | 普通模式界面 |
| `screenshot-2.png` | 1280x800 或 640x400 | 番茄模式界面 |
| `screenshot-3.png` | 1280x800 或 640x400 | 系统通知弹窗 |
| `screenshot-4.png` | 1280x800 或 640x400 | 统计信息展示 |

## 截图方法

### Mac 系统

1. 打开 Chrome 浏览器
2. 按 `Cmd + Option + I` 打开开发者工具
3. 按 `Cmd + Shift + M` 切换到设备模拟模式
4. 设置尺寸为 1280x800
5. 截图：`Cmd + Shift + 4`，然后按空格键选择窗口

### Windows 系统

1. 使用 Chrome 开发者工具调整窗口大小
2. 使用截图工具（如 Snipaste）截取 1280x800 区域

## 注意事项

- 截图中不要包含敏感个人信息
- 保持界面整洁，建议先完成几次活动以展示统计数据
- 使用中文界面截图（因为主要面向中文用户）
